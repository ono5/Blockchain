# Blockchain

## Install

```
npx create-next-app nft-marketplace --typescript
```

## Start

```
npm run dev
```

## Tailwind

[install](https://tailwindcss.com/docs/guides/nextjs)

```
npm install -D tailwindcss postcss autoprefixer
npm install @tailwindcss/forms
npx tailwindcss init -p
```
